**if sentence in shell script:**

```bash
if []; then
    #some codes here...
fi #结束if

if []; then
    #some codes here...
elif []; then
    #some codes here...
else
    #some codes here...
fi
```




* []中是判别式，和test用法一致。

**注：（主机特殊端口号）**

* 80:WWW
* 22:ssh
* 21:ftp
* 25:mail
* 111:RPC(远端程序调用)
* 631:CUPS（打印服务功能)
